# WemosRFID
RFID reader using Wemos D1 mini v2.2.0

Pin Layout
----------

|           |         | ESP8266       |
|-----------|---------|---------------|
|           | MFRC522 | Wemos D1 mini v2.2.0 |
| Signal    | Pin     | Pin           |
| RST/Reset | RST     | D0            |
| SPI SS    | SDA(SS) | D8            |
| SPI MOSI  | MOSI    | D7            |
| SPI MISO  | MISO    | D6            |
| SPI SCK   | SCK     | D5            |
