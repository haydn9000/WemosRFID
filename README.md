# WemosRFID
RFID reader using Wemos D1 mini v2.2.0

Pin Layout
----------

|           | ESP8266       |
|-----------|---------------|
|           | Wemos D1 mini v2.2.0 |
| Signal    | Pin           |
| RST/Reset | D0            |
| SPI SS    | D8            |
| SPI MOSI  | D7            |
| SPI MISO  | D6            |
| SPI SCK   | D5            |
